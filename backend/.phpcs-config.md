# PHPCS Configuration

## Standards
- **PSR-12**: Base coding standard
- **30+ Quality Rules**: Code analysis, naming, whitespace

## Key Rules

### Enforced
- `declare(strict_types=1)` required (except Kernel/bootstrap/fixtures)
- No deprecated/forbidden functions
- Proper indentation (4 spaces exact)
- No tabs, no trailing whitespace
- Logical operators (&&/|| not and/or)

### Exceptions
- Test methods: `test_snake_case` allowed (PHPUnit convention)
- Line length: No hard limit (120 guideline)

## Usage

```bash
# Check violations
make cs

# Auto-fix
make cs-fix

# Quality check (CS + PHPStan)
make quality
```

## Exit Codes
- 0: No errors
- 1: Errors found
- 2: Warnings only (non-blocking)

## Cache
- File: `.phpcs-cache`
- Speed: ~2x faster on repeated runs
- Parallel: 75 threads
